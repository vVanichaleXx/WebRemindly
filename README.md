# Remindly Web

Minimal React landing experience for Remindly with a full-screen animated intro and a small backend scaffold for future API work.

## Scripts

- `npm run dev` starts the React frontend and Express backend together.
- `npm run dev:frontend` starts only the Vite app.
- `npm run dev:backend` starts only the API.
- `npm run build` creates the frontend production bundle.

## Structure

- `frontend/src/components` contains reusable visual components.
- `frontend/src/config` keeps brand and animation constants.
- `frontend/src/styles` contains global and scene styles.
- `backend/src` contains the Express app, config, and routes.
